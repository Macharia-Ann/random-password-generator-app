# Password Generator App

A simple and interactive password generator built using HTML, CSS, and vanilla JavaScript.  
It generates strong random passwords with uppercase letters, lowercase letters, numbers, and symbols, and allows one-click copying to clipboard.

## Features

- Generates strong random passwords
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Fixed password length (12 characters)
- Ensures at least one character from each category
- Shuffles password for better randomness
- One-click copy to clipboard
- Visual feedback when password is copied

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
